
|regpack-1.54

Пак с .reg файлами для базовой оптимизации реестра Windows 10 и 11
Используйте папку «Restore» чтобы восстановить применённые .reg файлики
Файлы, заканчивающиеся на X требуют высшие права

Разработка учёных из TECHNOSHAHTA
https://discord.gg/mB6DprqmR9
https://t.me/TECHNOSHAHTA
https://github.com/donkrage/regpack


Информация о содержании:
|accessibility.reg - отключение настроек для людей с ограниченными возможностями
|appcompatibility.reg - отключение компонентов телеметрии и совместимости со старыми приложениями
|attachmentmanager.reg - отключение предупреждения безопасности при запуске файлов, скачанных с интернета
|backgroundapps.reg - отключение фоновых приложений (на Win 10 можно отключить в параметрах)
|cloudcontent.reg - отключение автоматической установки рекламных UWP приложений
|driversearching.reg - отключение автоматической установки драйверов
|edgeupdate.reg - отключение фоновых служб автообновления Microsoft Edge
|filesystem.reg - оптимизация файловой системы
|gamebar.reg - отключение Game Bar
|inspectre.reg - отключение Spectre и Meltdown защиты
|largesystemcache.reg - включение увеличенного системного кэша
|latestclr.reg - использование только последних версий .NET в приложениях
|maintenance.reg - отключение автоматического обслуживания
|oldphotoviewer.reg - возвращение старого просмотрщика фото
|priority.reg - увеличенный приоритет для игр
|responsiveness.reg - уменьшение процента используемых ресурсов для низкоприоритетных задач
|search.reg - отключение веб-поиска в меню поиска
|systemrestore.reg - отключение Точек восстановления и Службы теневого копиования
|uac.reg - отключение контроля учётных записей
|win10defenderX.reg - полное отключение Windows Defender, его служб и эксплойтов
|win11defenderX.reg - полное отключение Windows Defender, его служб и эксплойтов
|win11defsubsvcX.bat - определяет под-службу Defender с уникальным ID и отключает её. Нужно применять для полного отключения Windows Defender

Explorer
|3dedit.reg - убирает пункт «Изменить с помощью Paint 3D» из контекстного меню
|explorer.reg - небольшая настройка проводника
|foldernetworkX.reg - убирает папку «Сеть» в проводнике (на Win 11 доступно в настройках Проводника)
|menushowdelay.reg - отключение задержки показа контекстных меню
|win10folder3d.reg - убирает папку «Объемные объекты» в проводнике
|win10networkwizard.reg - отключение помощи настройки при подключении новой сети
|win10shareitem.reg - убирает пункт «Отправить» из контекстного меню
|win10showsecondsinsystemclock.reg - вывод секунд в системные часы, которые отображаются на панели задач
|win11contextmenu.reg - классическое контекстное меню
|win11shareitem.reg - убирает пункт «Отправить» из контекстного меню
