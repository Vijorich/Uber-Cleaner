|regpack 1.36

Рег пак с рег файликами для точечной оптимизации реестра Windows

Разработка учёных из TECHNOSHAHTA
https://discord.gg/mB6DprqmR9

Информация о содержании:

Пункт просто применить рекомендуемые .reg файлы содержит в себе:

1)Абсолютно все .рег файлы из пунктов windows 11 и windows 10 only (Применяются сами в зависимости от вашей версии windows)

2)Общие для всех версий .reg файлы
|inspectre - отключение spectre и meltdown защиты, аналог утилиты inspectre
|uac - отключение контроля учётных записей
|maintenance - отключение maintenance (автоматическое обслуживание)
|attachmentmanager - отключение предупреждения безопасности при запуске файлов, скачанных с интернета
|telemetry - базовое отключение телеметрии
|cloudcontent - отключение автоматической установки uwp приложений (нетфликс картишки конфетки яндекс и подобный кал)
|driversearching - отключение установки драйверов с центра обновлений
|appcompability - отключение компонентов телеметрии и совместимости со старыми приложениями
|backgroundapps - отключение фоновых приложений (на вин 10 можно отключить в параметрах, на вин 11 с помощью реестра)
|menushowdelay - убирает задержку показа менюшек
|regionalformats - установка нормальных форматов даты, времени и метрической системы (Monday | Friday, 14 April, 2017 | 14.04.17 | 09:40:07)
|search - отключение веб-поиска в меню поиска
|latestclr - использование только последних версий .NET в приложениях (чтобы не устанавливать старые версии)
|gamedvr - отключение гейм бара без выпиливания важных функций игровых сервисов
|fse - глобальное отключение оптимизации во весь экран
|explorer - небольшая настройка проводника (описание настроек читайте в самом рег файле)
|filesystem - оптимизация файловой системы
|networkfolder X - удаление папки "Сеть" в проводнике
|systemprofile - уменьшение процента используемых ресурсов для низкоприоритетных задач
|systemrestore - отключение точек восстановления

Пункт применить каждый точечно (для любой версии шиндус) содержит в себе:

1) Отключить антивирус, телеметрию и прочую защиту:
|inspectre - отключение spectre и meltdown защиты, аналог утилиты inspectre
|uac - отключение контроля учётных записей
|maintenance - отключение maintenance (автоматическое обслуживание)
|attachmentmanager - отключение предупреждения безопасности при запуске файлов, скачанных с интернета
|telemetry - базовое отключение телеметрии
|autoupdate - отключение автообновлений, рекомендуется использовать после установки всех обновлений

2) Отключить все автообновления:
|autoupdate - отключение автообновлений, рекомендуется использовать после установки всех обновлений
|cloudcontent - отключение автоматической установки uwp приложений (нетфликс картишки конфетки яндекс и подобный кал)
|driversearching - отключение установки драйверов с центра обновлений

3) Отключение компонентов совместимости
|appcompability - отключение компонентов телеметрии и совместимости со старыми приложениями

4) Отключение фоновых приложений
|backgroundapps - отключение фоновых приложений (на вин 10 можно отключить в параметрах, на вин 11 с помощью реестра)

5) Оптимизация файловой системы
|filesystem - оптимизация файловой системы
|explorer - небольшая настройка проводника (описание настроек читайте в самом рег файле)
|networkfolder X - удаление папки "Сеть" в проводнике

6) Включает функцию largesystemcache
|largesystemcache - включает функцию largesystemcache, может повысить производительность

7) Отключение гей бара
|gamedvr - отключение гейм бара без выпиливания важных функций игровых сервисов

8) Возвращение старого просмотрщика фото
|backoldphotoviewer - возвращение старого просмотрщика фото

9) Убрать задержку показа менюшек
|menushowdelay - убирает задержку показа менюшек

10) Установить нормальный форматы даты, времени и метрическую систему 
|regionalformats - установка нормальных форматов даты, времени и метрической системы (Monday | Friday, 14 April, 2017 | 14.04.17 | 09:40:07)

11) Отключить веб поиск в меню поиска
|search - отключение веб-поиска в меню поиска

12) Уменьшение процента используемых ресурсов для лоу-приорити задач
|systemprofile - уменьшение процента используемых ресурсов для низкоприоритетных задач

13) Отключение точек восстановления
|systemrestore - отключение точек восстановления

14) Глобальное отключение оптимизации во весь экран
|fse - глобальное отключение оптимизации во весь экран

15) Использование только последних версий .NET
|latestclr - использование только последних версий .NET в приложениях (чтобы не устанавливать старые версии)

16) Отключение всех эксплойтов, кроме CFG
|exploitscfg - отключение всех эксплойтов, кроме CFG (требуется для работы античита Kalorant). используйте после отключения дефендера

17) Отключить службы автообновления и фоновых процессов Edge браузера
|edge - отключение служб автообновления и фоновых процессов Edge браузера

18) Отключить телеметрию NVIDIA
|nvtelemetry - отключение телеметрии NVIDIA

19) Отключение всех эксплойтов
|exploits - отключение всех эксплойтов

20) Поставить префетч в значение 3
|prefetcher 2 - префетчер на двочеку и включение службы SysMain

Windows 10 Only
|w10 notifications - полное отключение уведомлений и центра уведомлений
|w10 priority - увеличенный приоритет для игр, улучшает производительность (для вин 10)
|w10 share item - удаление пункта "Отправить" из контекстного меню
|w10 3d objects - удаление папки "Объемные объекты" в проводнике (на вин 11 этой папки нет)
|w10 defender X - полное отключение дефендера, smartscreen, эксплойтов
|w10 showsecondsinsystemclock - вывод секунд в системные часы, которые отображаются на панели задач
|w10 newnetworkwindowoff - отключение уведомления при подключении новой сети

Windows 11 Only
|w11 context menu fix - фикс уёбищных контекстных меню виндовс 11
|w11 defender X - полное отключение дефендера, smartscreen, эксплойтов
|w11 notifications - полное отключение уведомлений
|w11 priority - увеличенный приоритет для игр, улучшает производительность (для вин 11)
|w11 share item - удаление пункта "Отправить" из контекстного меню
